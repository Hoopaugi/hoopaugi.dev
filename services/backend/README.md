# hoopaugi.dev backend

## Commands

Lint code

    npm run lint

Lint code and fix errors

    npm run lint:fix

Run tests

    npm run test

Build project

    npm run build

Run in development mode

    npm run start:dev

Run in production mode

    npm run start:prod

## .env

Example .env variables in `example.env` in backend root

|Variable|Required|Default|Description|
|-|-|-|-|
|PORT||5000|Port the backend runs on|
